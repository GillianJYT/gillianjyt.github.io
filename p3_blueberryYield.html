Blueberry Yield (Mar 2024)

Overview

Using this Kaggle competition, we wanted to predict the yield of wild blueberries. To compare and contrast machine learning methods, the problem was solved using 5 different models, evaluated using Mean Absolute Error (MAE). We also calculated each models' Mean Squared Error (MSE), Mean Absolute Percentage Error (MAPE) and R2 Score.

My Contribution and Takeaways

I worked on the Multilayer Perceptron (MLP) Neural Network. Before feeding the data into the model, I normalized the data using StandardScalar, and then tuned the hyperparameters using grid search.

Possibilities of Further Development

This project itself is completed, so no further development can be made. For my personal development, I would like to practice machine learning and create more complex neural networks.
