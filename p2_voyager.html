Voyager (2023-2024)

Overview

Voyager is an online vacation itinerary scheduling and suggestion tool whose mission is to make travel planning simpler and collaborative. Given a few parameters, the user receives a baseline itinerary they can edit as they please, and the suggestion tool is always available to help schedule events. All voyages can be shared with friends and family so that everyone can make the most of the vacation. By providing this service free of charge and readily available online, anyone can plan their dream vacation.

My Contribution and Takeaways

I was primarily the Project Manager in this project. I managed scheduling and kept track of upcoming deadlines, handled documentation and paperwork, and ensured that the project goals were being met.

I also worked on the UX design of the appliation. This involved basic sketches, wireframing, and using Figma. I led the design from initial low-fidelity sketches to creating high-fidelity mockups. I was able to practice using Figma through incorporating more interactive components in the mockups.

Possibilities of Further Development

Given more time, I would love to conduct more UX tests. That includes user tests to assess the ease of use, and more analytical tests for accessability needs.
